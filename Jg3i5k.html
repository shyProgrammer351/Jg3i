 판에 해당하는 컨테이너 클래스로 버튼이나 체그박스 등 다른 요소를 포함 가능  

 텍스트나 이미지 아이콘을 포함할 수 있는 컴포넌트 클래스

 다른 객체를 담을 수 있는 클래스  

 격자 모양으로 컴포넌트를 배치하는 클래스  

 여러 컴포넌트를 쌓고 한번에 하나씩 제시하는 클래스  

 컴포넌트를 배치 형태를 지정하는 메서드  

 카드레이아웃에서 첫번째 컴포넌트로 이동하는 메서드  

 카드레이아웃에서 이전 컴포넌트로 이동하는 메서드

 버튼 클릭이나 타이머 활성화 등의 이벤트  

 버튼 클릭이나 타이머 활성화 등의 이벤트 발생시 해당 메서드를 호출하는 리스너  

 버튼 등의 컴퍼넌트에 리스너를 추가하는 메서드  

 버튼 클릭이나 타이머 활성화 등의 이벤트 발생시 호출하는 메서드  

 이벤트를 발생시킨 객체를 가져오는 메서드
