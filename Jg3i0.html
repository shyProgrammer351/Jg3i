Component, Event, Listener의 관계

Component   사용자 인터페이스를 구성하는 기본적인 요소(예: 프레임, 패널, 버튼, 체크박스, 라디오 버튼 등)

Event   다양한 사용자 인터페이스 액션(예: 버튼 클릭, 키 입력, 마우스 움직임 등)

Listener   이벤트가 발생했을 때 이를 감지하고 특정 동작을 수행하는 인터페이스

Component에 Event가 발생하면 등록한 Listener에서 method 호출

addActionListener   버튼 클릭이나 타이머 활성화 등의 이벤트가 발생할 때까지 대기할 리스너를 등록

addItemListener   체크박스, 라디오 버튼 등에서 항목 변경 이벤트가 발생할 때까지 대기할 리스너를 등록

addMouseListener   마우스에서 클릭, 드래그 등의 이벤트가 발생할 때까지 대기할 리스너를 등록

addKeyListener   키보드 입력 이벤트가 발생할 때까지 대기할 리스너를 등록

addWindowListener   윈도우에서 창 열기, 창 닫기 등의 이벤트가 발생할 때까지 대기할 리스너를 등록

MouseAdapter   MouseListener 인터페이스의 모든 메서드를 구현한 클래스
MouseAdapter를 상속하면 수정이 필요한 일부 메서드만 재정의 가능

KeyAdapter   KeyListener 인터페이스의 모든 메서드를 구현한 클래스
KeyAdapter를 상속하면 수정이 필요한 일부 메서드만 재정의 가능

WindowAdapter   WindowListener 인터페이스의 모든 메서드를 구현한 클래스
KeyAdapter를 상속하면 수정이 필요한 일부 메서드만 재정의 가능

다양한 Component

JFrame   틀에 해당하는 최상위 컨테이너 클래스로 운영체제마다 다른 틀 모양 제공

JPanel   판에 해당하는 컨테이너 클래스로 버튼이나 체그박스 등 다른 요소를 포함 가능

JLabel   텍스트나 이미지 아이콘을 포함할 수 있는 컴포넌트 클래스

setText   객체에 출력할 텍스트의 내용 설정하는 메서드

setFont   해당 객체에 출력할 텍스트의 글자모양 설정 하는 메서드

JButton   클릭할 수 있는 버튼 클래스

setEnabled   전달받은 true, false에 따라 버튼을 활성화하거나 비활성화하는 메서드

Timer   생성시 전달받은 시간과 리스너를 바탕으로, 일정 시간후 해당 리스너를 활성화시키는 클래스

start   타이머를 시작하는 메서드

stop   타이머를 종료하는 메서드

ImageIcon   이미지 아이콘 컴포넌트 클래스

setIcon   레이블을 전달받은 이미지로 설정하는 메서드

repaint   컴포넌트를 다시 그리는 메서드로, 내부적으로 paintComponent를 호출
