문제

숫자를 입력받아 홀수인지 짝수인지 맞추는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요

알고리즘

프로그램시작 
   임의의 수 숨기기 
   임의의 수 맞추기 
      숫자 입력 받기 
         입력값이 제한범위 벗어나면 종료 
      비교 결과 출력 
         첫째 임의의 수를 맞추면 Success!! 출력 후 종료 
         둘째 임의의 수가 입력값보다 크면 Up! 출력 
         셋째 임의의 수가 입력값보다 작으면 Down! 출력 
   계속 반복할지 확인 
프로그램 종료

프로그램 코드

	import javax.swing.JOptionPane;

5				if ( ( user < 1 ) || ( 100 < user ) ) {

6					JOptionPane.showMessageDialog( null, "제한범위를 벗어났습니다" );

7					continue;

				}

				// 첫째 임의의 수를 맞추면 Success!! 출력 후 종료  

8				else if ( number == user ) {

11				else if ( number > user ) {

12					JOptionPane.showMessageDialog( null, "Up!" );
